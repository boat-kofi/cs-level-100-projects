# Simple calculator using while loop 

## Description
A simple calculator built in C to perform addition, subtraction, multiplication and division

## Features
- Add two numbers
- Subtract two numbers
- Multiply two numbers
- Divide two numbers

## Built with
- C
- Dev-C++
- Git for version control
- Github for project history

## Project setup/How to Run
- Ensure you have Git installed
- Copy URL
- Open your terminal(Git bash on Windows, Terminal on Mac/Linux)
- Navigate to your preferred directory
- Run this command "git clone [copied URL]"
- Open the folder in Dev-C++
- Compile and run ifelsesimplecal.c
